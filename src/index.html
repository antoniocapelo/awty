Add one class when visible

How?

  • Add data-awty="classname" on the html element
  • voilá :)

Add many classes when visible

How?

  • Add data-awty="classname1,classname2" on the html element
  • voilá

Add classes when element is centered

How?

  • Add data-awty="classname"
  • data-awty-mode="center"

Or remove classes when visible

How?

  • Add data-awty="classname" on the html element
  • data-awty-remove="true" to use remove mode

Add class when a % of element is visible

How?

  • Add data-awty="classname" on the html element
  • Define mode data-awty-mode="peek"
  • Define percentage data-awty-space="0.35"

Or Add classes after a certain margin

How?

  • Add data-awty="classname" on the html element
  • Define mode data-awty-mode="peek"
  • Define percentage data-awty-space="0.35"

And that's it :)
